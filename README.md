     Gestor de Gastos

Aplicación web para gestionar gastos personales de forma simple.

     Demo
https://proyecto-gestor-de-gastos.netlify.app/

     Funcionalidades
- Agregar gastos
- Eliminar gastos
- Calcular total automáticamente
- Guardado en localStorage
- Botón para limpiar todo

    Tecnologías
- HTML
- SASS
- JavaScript
- Gulp

    Objetivo
Proyecto realizado como práctica de desarrollo frontend para portfolio.
